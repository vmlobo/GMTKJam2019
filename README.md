You only have Juan Clip, your goal is to survive.

Instructions:
Kill as many enemies as you can using only one clip.
Enemies have a chance to drop ammo.

Controls: 
'Mouse 1' - Fire
'Space' or 'Mouse 2' - pickup ammo
